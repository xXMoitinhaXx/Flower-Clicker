🌸 Flower Clicker
Um jogo incremental feito em Python + Tkinter, com sistema de níveis, upgrades, rebirths, animações e efeitos visuais.

🎮 Sobre o Jogo
Flower Clicker é um clicker game onde o jogador evolui uma flor através de cliques, upgrades e renascimentos.
À medida que progrides, a flor transforma‑se, o fundo muda, novas mecânicas aparecem e o jogo torna‑se cada vez mais dinâmico.

O objetivo é simples:
clicar, evoluir, renascer e repetir.

✨ Funcionalidades Principais
🌼 Evolução da Flor
A flor muda de aparência conforme o nível.

Após o nível 250, transforma‑se numa árvore.

Nível especial: 143 desbloqueia uma flor única.

💰 Sistema de Pontos
Clique normal: ganha pontos base.

Upgrade “MORE POINTS UPD”: aumenta os pontos por clique.

Textos animados mostram os pontos ganhos.

🔁 Sistema de Rebirth
Rebirth 1: desbloqueia pontos por clique adicionais.

Rebirth 2: desbloqueia folhas caindo e pontos por folha apanhada.

🍃 Folhas e Estrelas Cadentes
Estrelas cadentes aparecem aleatoriamente no céu.

Folhas caem quando o jogador atinge Rebirth 2.

Clicar nas folhas dá pontos extra.

🎵 Música de Fundo
Música em loop usando winsound.

💾 Sistema de Save Automático
Salva automaticamente a cada 30 segundos.

Usa o módulo personalizado bib_saves.
